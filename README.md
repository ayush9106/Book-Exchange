# 📚 Book Exchange Web Application

## 📌 Project Overview
A web-based application to buy, sell, and exchange books between users.

## 🛠 Technologies Used
- JSP & Servlets (Frontend + Controller)
- JDBC (Database Connectivity)
- MySQL (Database)
- Apache Tomcat (Server)

## 🧱 MVC Architecture
- Model → Database + Java classes
- View → JSP pages
- Controller → Servlets

## 🚀 Features
- User Registration & Login
- Add / View Books
- Send Request
- Accept / Reject Requests
- Payment Page (UI)

## 🗄 Database
Database Name: `book_exchange`

## ▶️ How to Run
1. Import project into Tomcat
2. Start server
3. Open: `http://localhost:8080/BookExchange`
